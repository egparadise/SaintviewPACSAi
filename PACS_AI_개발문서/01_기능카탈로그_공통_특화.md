## 1. 공통 기능 (Core — 반드시 구현)

세 제품 모두(또는 다수 + 영상) 공통으로 제공하는, PACS 뷰어의 **최소 필수 기능 집합**. saintvidw pacs ai의 1차 개발 범위로 삼는다.

### 1.1 인증·세션
- ID/PW 로그인, 비밀번호 변경, 세션 유지.

### 1.2 워크리스트 & 검색
- 조건 없는 검색(검사일 최근순), 조건 검색(필터 콤보박스/직접입력), 빠른 검색(키워드+Enter), 기간 검색(시작·종료일 캘린더).
- 검색 조건 삭제(Erase / 우클릭), 검색 바로가기 버튼 등록·실행.
- 검색 필터 항목 ON/OFF 확장 설정.
- Study/Series/Instance 트리 확장, 컬럼 정렬(▲▼), 컬럼 위치 드래그 이동, 환자ID 클릭 필터.
- 상태표시줄: Total/Displayed/Selected, 저장용량.

### 1.3 뷰어 열기 & 레이아웃
- 열기 모드: **View**(교체) / **Add View**(추가) / **Stack View**(중첩).
- 더블클릭 열기, 썸네일 드래그앤드롭 표시.
- 화면 분할: Study Layout / Series Layout / Image Layout (1x1~2x2 등 NxN).
- 썸네일/미리보기 바(위치 좌·우·상 전환).

### 1.4 영상 조작 (Manipulation)
- **W/L 윈도잉**(드래그: 상하=Center, 좌우=Width) + **프리셋**(단축키 0~9, Default/Full Dynamic).
- Pan(이동), Zoom(확대/축소), 돋보기(국소확대), 화면맞춤(Fit), Actual Pixel.
- 회전(90°/180°/임의), 좌우반전·상하반전, 색상반전, 의사/유사색상 매핑.
- 초기화(Reset: W/L·Pan·Zoom·Rotate·Flip 등), 새로고침.

### 1.5 계측 & 주석 (Annotation/Measurement)
- 선/길이, 화살표, 사각형, 원/타원, 다각형, 자유곡선, 각도, Cobb's angle.
- 텍스트, 메모, 마킹, ROI 측정값(면적·평균 등).
- 주석 이동/변형/삭제(선택삭제·전체삭제), 주석 저장.

### 1.6 판독 (Reporting)
- 판독판: Findings / Impression / Comments 입력.
- 템플릿 삽입, Std Report(Normal Form) 등록·재사용.
- 상태: Draft/Final, Confirm(승인), Suspend(일시중단), 이력(History).
- 저장(덮어쓰기/비덮어쓰기), 인쇄, PDF 변환·내보내기.

### 1.7 출력·데이터
- 인쇄(이미지 전용 / 이미지+판독문), 내보내기(DCM/JPG/PNG/BMP), 전송(C-STORE).
- DICOM Header 조회, 캡처(클립보드), 재생(Cine).

### 1.8 설정·단축키
- 툴바/검색/목록 항목 ON·OFF·순서·이름 변경(사용자 설정).
- 키보드 단축키 체계 + 단축키 도움말.

## 2. 특화 기능 (Differentiator — 선택·차별화 구현)

제품별 강점. saintvidw pacs ai가 **AI로 재해석하거나 차별화**할 수 있는 영역.

### 2.1 INFINITT — 3D·고급 가시화 강점
- **3D 엔진**: MPR(다평면 재구성), MIP, Volume Rendering, Virtual Endoscopy, 3D Segmentation.
- **Crosslink / Auto Sync / Sync With Other Exams / Scout Line / All Lines** — 시리즈·검사 간 위치 동기화.
- **Hanging Protocol(글로벌)**, **Windowing Preset 핫키**, web 기반 + JPEG2000 압축 전송, IHE 상호운용.

### 2.2 UBPACS-Z / ZeTTA — 워크플로우·판독 생산성 강점
- **음성 판독(녹음/재생)** — STT/AI 초안화의 핵심 후보.
- **커스터마이즈 워크리스트**(최대 10개 컴포넌트: Search Wizard, Related Study List 1·2, Reference/Comment/Thumbnail/Report/Memo Window).
- **Filter Property**: `Keyword` / `Keyword%` / `%Keyword%` / `!=` 연산자, Add Keyword·Clear History.
- **열기 모드 확장**: Advance View, Key Image View, **Study With Open**(관련검사 동시 오픈).
- **판독 확장**: Merge(묶음판독), Copy(판독문 복사), Related(참조), Suspend, Std Report.
- **특수 계측**: CT Ratio(심흉비), Limb LD(다리길이), Q-Angle, Spine Label, Calibrate, Shutter(영역보기).
- **영상처리**: Sharpen/Average 필터(3단계), 위치기준 레벨링(마우스 위치 기억).
- **External Link 1/2/3**(OCS·외부 프로그램 호출), Advanced Layout, CD Burn.

### 2.3 SonicPACS — 데이터 관리·동기화 강점 (Server+Web 통합)
- **상세 동기화(Sync)**: 스크롤(slice position/ratio/normal), 확대(mm/px/%), 회전·반전·W/L·색상 동기.
- **비교(Compare)**: 현재검사 ↔ 최근검사 **modality+촬영부위 자동 매칭**.
- **MPR**, **유사색상**(Grayscale/Hot Iron/Hot Metal Blue/PET/PET 20 Step), **이미지 보간**(Nearest/Bilinear/Bicubic).
- **익명화**(환자ID·이름·DOB ****), **DICOM 정보**, **디스플레이 토글**(주석/오버레이/방향/눈금자/참조선/스크롤바).
- **데이터 라이프사이클**: 추가(JPEG/PNG/BMP/PDF→DICOM), 수정(Study/Series/Instance), 삭제, 전송, **내보내기(ISO/ZIP/DICOMDIR + 무료 뷰어 첨부)**, 가져오기.
- **리포트**: 템플릿, Draft/Final + 덮어쓰기 옵션, 이력, PDF.
- **풀 키보드 단축키 체계**(기능/주석/마우스드래그/네비게이션/Cine/텍스트/화면 토글).

## 3. saintvidw pacs ai 적용 관점 — AI 차별화 후보

기존 PACS가 "수동 도구" 중심이라면, AI 에이전트형 PACS는 다음을 **자동·대화형**으로 끌어올린다.

| 기존 수동 기능 | AI 에이전트화 방향 |
|----------------|--------------------|
| W/L 프리셋 수동 선택 | 모달리티·부위 인식 → **자동 최적 W/L** 적용 |
| 수동 계측(길이/각도/CTR) | **자동 계측·라벨링**(심흉비·척추각 자동 산출) |
| Prior 비교 수동 매칭 | **자동 prior 정렬 + 변화 하이라이트** |
| Findings 수동 타이핑 | **AI 판독 초안 생성** + 단축키로 확정 |
| Hanging Protocol 규칙 설정 | 검사 맥락 기반 **자동 행잉** |
| 검색어 입력 | **자연어 질의**("지난주 흉부 CT 미판독") |
| 주석 수동 작성 | 검출 병변 **오버레이 자동 주석** |
